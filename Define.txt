# Tổng quan
- File được dùng để mô tả tổng thể mô hình backend được xây dựng.
- Kiến thức: Spring Boot Web, JPA, Lombok, Hibernate, Mô hình MVC Spring

# Cấu trúc
- Gồm 3 tầng (tự custom lại theo ý thích của người sáng tạo):
	+ API (apis): Cung cấp đường dẫn cho các chương trình gọi tới
	+ Xử lý (services): Xử lý nội dung được trả về
	+ Lấy dữ liệu (repossitories): Lấy dữ liệu dưới Database
- utils: Cung cấp các công cụ mà cả hệ thống có thể sử dụng
- models: Cung cấp mô hình và dùng để khởi tạo entity dưới database
- configs: Thông tin cho dự án

# Giải thích api:
- Liên quan đến truy cập
	+ /api/v1/account/login: Đăng nhập -> JSON {'email':'?', 'password': '?'}
	+ /api/v1/account/register: Đăng ký -> JSON {'email':'?', 'password': '?'}
	+ /api/v1/account/forget: Quên mật khẩu -> JSON {'email':'?'}
